API Key Generator

HOW TO?

1. Generate API Key

Click generate to generate a new 32-character API key to be used as the authorization token.
Generating doesn't necessarily set the key, it just generates it.

2. Enter the Passcode and set API Key

Enter the passcode to set the API key as the active token.
Every request made to the server will now be authenticated with the API key.
Setting the key also copies it to the clipboard.

3. Use the API Key to send requests

Finally, store the API key in a safe place, as it is the only way to authenticate with the server.
In case you lose the key or someone else steals it, you can generate a new one and set it as the active token.
Every future request made to the server will be authenticated with the new key.
